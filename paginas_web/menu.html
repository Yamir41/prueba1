Reflexión Personal

A lo largo de este curso de Páginas Web, he construido un aprendizaje sólido y progresivo sobre el desarrollo web. En el primer parcial, comencé con la teoría fundamental identificando las partes esenciales de una página web y diseñando mis primeras maquetas visuales, para luego adentrarme en el lenguaje CSS mediante el uso de selectores básicos y avanzados. En el segundo parcial, puse en práctica la maquetación estructural completa utilizando etiquetas semánticas de HTML y dándole formato a los menús de navegación interactivos mediante propiedades de CSS avanzadas. Finalmente, en este tercer parcial, consolidé mi portafolio comprendiendo el ciclo cliente-servidor mediante peticiones HTTP, aplicando las reglas internacionales de la W3C, implementando atributos de accesibilidad web (WAI-ARIA) para hacer el sitio inclusivo, optimizando los meta tags para SEO y realizando el lanzamiento final del proyecto a un hosting en vivo. Este portafolio representa no solo mi esfuerzo técnico, sino mi evolución en la creación de soluciones web funcionales y profesionales.
